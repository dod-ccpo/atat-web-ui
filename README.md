# atat-web-ui
ATAT Web - Front end

## global install

```
 nvm use lts/fermium 
 npm install -g @vue/cli
 npm i -g @vue/cli-service-global
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
